# Heritage Operations Center

Enterprise Operations Portal for Heritage Environments.

## Features

- Dashboard
- Restart Center
- Deployment Center
- Repository Manager
- Database Manager
- SSH Console
- JBeam
- Reports
- Utilities
- History
- Settings

## Technology

- Python
- FastAPI
- HTML
- CSS
- JavaScript
- Paramiko
- Oracle
- Pandas

## Start

```bash
pip install -r requirements.txt
uvicorn app:app --reload
```

Open:

http://127.0.0.1:8000
